# 가상환경(venv) 명령어 

- 가상환경 생성 

    > ```
    > python -m venv 가상환경이름
    >```
<br>

- 가상환경 활성화 

    > ```
    > source 가상환경이름/Scripts/activate
    >```

<br>

- 가상환경 비활성화 

    > ```
    > deactivate
    > ```

<br>

- 패키지 설치 명령어

    > ```
    > pip install <패키지>
    > # ex) pip install numpy
    >```

<br>

- 가상환경 정보 저장 명령어

    >```
    >pip freeze > requirements.txt 
    >```

<br>

- 버전 정보 파일로 설치하는 명령어

    > ```
    > pip install -r requirements.txt
    > ```
