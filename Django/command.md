# Django 명령어 

- 장고 설치 

    > ```
    > pip install django
    > ```

<br>

- 프로젝트 시작

    > ```
    > django-admin startproject [project name]
    > ```

<br>

- 앱 만들기

    > ```
    > python manage.py startapp [app name]
    > ```

<br>

- models.py 변경사항 적용

    > ```
    > python manage.py makemigrations
    > ```

<br>

- models.py 수정후 데이터 매핑

    > ```
    > python manage.py migrate
    > ```

<br>


- 서버 실행

    > ```
    > python manage.py runserver
    > ```

<br>
